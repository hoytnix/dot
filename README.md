Openbox Commands:
-----------------

Openbox:

`Super+Space`: Openbox menu.

Screenshots:

`PrtSc`: select a portion of the screen.

`Alt+PrnSc`: full-screen.

`Ctrl+PrnSc`: active window.

`Alt+Ctrl+PrnSc`: select a window.

Aerosnap:

`Super+DirectionalKey`: Resize window.

`Super+Alt+Left/Right`: Aerosnap.
